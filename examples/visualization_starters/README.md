# Visualization Starters

This folder contains starter artifacts for the optional post-run `graphics` agent.

Current starter:

- `google_sankey_template.html` - standalone Google Charts Sankey example styled for repo-level research deliverables

Suggested use:

1. Run the pipeline and inspect `output/summary_stats.json`, `output/coded_posts.csv`, and `output/summary_report.md`.
2. Replace the sample rows in the starter with real values derived from those files.
3. Export both an interactive HTML file and a static SVG or PNG preview into `output/visualizations/`.

Use `docs/GRAPHICS_AGENT.md` for chart-selection rules and artifact conventions.
